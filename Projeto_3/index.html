Introdução

Java é uma linguagem de programação versátil e orientada a objetos amplamente utilizada para construir aplicativos robustos e escaláveis. É conhecida por sua filosofia "write once, run anywhere".

Esta seção fornecerá uma introdução aos conceitos fundamentais da programação em Java.

Variáveis

Em Java, as variáveis são usadas para armazenar e gerenciar dados. Elas têm um tipo de dado específico, e o tipo deve ser declarado explicitamente.

Exemplo:

int idade = 25;
String nome = "João";

Tipos comuns de variáveis:

  • int
  • double
  • boolean
  • String
  • char
Métodos

Métodos (funções) em Java são blocos de código que realizam uma tarefa específica. Eles são definidos dentro de classes e seguem os princípios da programação orientada a objetos.

Exemplo:

public void cumprimentar(String nome) {
 System.out.println("Olá, " + nome + "!");
}
cumprimentar("Alice");
Estruturas de Repetição

Java suporta vários tipos de estruturas de repetição, como for, while e do-while. Elas são usadas para iterar sobre sequências ou executar um bloco de código repetidamente.

Exemplo:

for (int i = 0; i < 5; i++) {
 System.out.println(i);
}
Estruturas de Dados

Java fornece um conjunto rico de estruturas de dados, incluindo arrays, listas e mapas. Essas estruturas são essenciais para organizar e manipular dados de maneira eficiente.

Exemplo:

List minhaLista = new ArrayList<>();
minhaLista.add("Java");
System.out.println(minhaLista.get(0));
